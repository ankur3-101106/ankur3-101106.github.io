Generalised ElGamal Public Key Cryptosystem​

Setup (General)​

  • Cyclic group:

    GGG

  • Generator:

    α\alphaα

  • Private key:

    aaa

  • Public key:

    β=αa\beta = \alpha^aβ=αa

  • Public:

    (G,α,β)(G,\alpha,\beta)(G,α,β)

Encryption​

Choose random:

kkk

Ciphertext:

y1=αky_1 = \alpha^ky1​=αk y2=x⋅βky_2 = x \cdot \beta^ky2​=x⋅βk

Decryption​

x=y2⋅(y1a)−1x = y_2 \cdot (y_1^a)^{-1}x=y2​⋅(y1a​)−1

Core Hard Problem​

β=αa⇒find a\beta = \alpha^a \Rightarrow \text{find } aβ=αa⇒find a

(Discrete Log Problem)

Elliptic Curve Version​

  • Group:

    E(Zp)E(\mathbb{Z}_p)E(Zp​)

  • Operation: point addition

Keys​

  • Generator point:

    α\alphaα

  • Private key:

    aaa

  • Public key:

    β=aα\beta = a\alphaβ=aα

Encryption (ECC)​

Choose random:

kkk y1=kαy_1 = k\alphay1​=kα y2=x+kβy_2 = x + k\betay2​=x+kβ

Decryption​

x=y2−ay1x = y_2 - a y_1x=y2​−ay1​

  • p=11p = 11p=11

  • Curve:

    y2≡x3+x+6(mod11)y^2 \equiv x^3 + x + 6 \pmod{11}y2≡x3+x+6(mod11)

  • Generator:

    α=(2,7)\alpha = (2,7)α=(2,7)

    ECDLP\text{ECDLP}ECDLP

Lecture 32:
Chinese Remainder Theorem​

Linear Equations (mod p)​

Given:

4x+7y≡9(mod13)4x + 7y \equiv 9 \pmod{13}4x+7y≡9(mod13) 5x+2y≡3(mod13)5x + 2y \equiv 3 \pmod{13}5x+2y≡3(mod13)

Steps​

  • Multiply by inverses:
4−1≡10(mod13)4^{-1} \equiv 10 \pmod{13}4−1≡10(mod13) 5−1≡8(mod13)5^{-1} \equiv 8 \pmod{13}5−1≡8(mod13)

Transform​

x+5y≡12(mod13)x + 5y \equiv 12 \pmod{13}x+5y≡12(mod13) x+3y≡11(mod13)x + 3y \equiv 11 \pmod{13}x+3y≡11(mod13)

Subtract​

2y≡1(mod13)2y \equiv 1 \pmod{13}2y≡1(mod13) y≡7(mod13)y \equiv 7 \pmod{13}y≡7(mod13)

Substitute​

x=12−5y≡3(mod13)x = 12 - 5y \equiv 3 \pmod{13}x=12−5y≡3(mod13)

Solution​

x=3,y=7x = 3,\quad y = 7x=3,y=7

Chinese Remainder Theorem (2 equations)​

Given:

x≡a(modm)x \equiv a \pmod{m}x≡a(modm) x≡b(modn)x \equiv b \pmod{n}x≡b(modn)

Condition:

gcd⁡(m,n)=1\gcd(m,n)=1gcd(m,n)=1

General Form​

x=a+tmx = a + tmx=a+tm

Substitute:

a+tm≡b(modn)a + tm \equiv b \pmod{n}a+tm≡b(modn)

Solve for ( t )​

t≡m−1(b−a)(modn)t \equiv m^{-1}(b-a) \pmod{n}t≡m−1(b−a)(modn)

Final Solution​

x=a+tm(modmn)x = a + tm \pmod{mn}x=a+tm(modmn)

Example​

x≡3(mod5)x \equiv 3 \pmod{5}x≡3(mod5) x≡5(mod7)x \equiv 5 \pmod{7}x≡5(mod7)

Compute​

5−1≡3(mod7)5^{-1} \equiv 3 \pmod{7}5−1≡3(mod7) t=3(5−3)≡6(mod7)t = 3(5-3) \equiv 6 \pmod{7}t=3(5−3)≡6(mod7)

Result​

x=3+5⋅6=33x = 3 + 5 \cdot 6 = 33x=3+5⋅6=33 x≡33(mod35)x \equiv 33 \pmod{35}x≡33(mod35)

General CRT (r equations)​

Given:

x≡ai(modmi)x \equiv a_i \pmod{m_i}x≡ai​(modmi​)

Condition:

gcd⁡(mi,mj)=1\gcd(m_i,m_j)=1gcd(mi​,mj​)=1

Define​

M=∏miM = \prod m_iM=∏mi​ Mi=MmiM_i = \frac{M}{m_i}Mi​=mi​M​ yi=Mi−1(modmi)y_i = M_i^{-1} \pmod{m_i}yi​=Mi−1​(modmi​)

Solution​

x=∑aiMiyi(modM)x = \sum a_i M_i y_i \pmod{M}x=∑ai​Mi​yi​(modM)

Key Properties​

  • Unique solution:

    mod  (∏mi)\mod \left(\prod m_i\right)mod(∏mi​)

  • Works only if moduli are pairwise coprime

Key Insight​

  • Break big modulus into smaller ones
  • Solve separately
  • Combine using CRT

Final Note​

  • Used in:

    • RSA optimization
    • Cryptographic computations

Lecture 33:
Rabin Cryptosystem​

Quadratic Residue​

Definition:

a is quadratic residue mod p  ⟺  ∃y: y2≡a(modp)a \text{ is quadratic residue mod } p \iff \exists y:\ y^2 \equiv a \pmod{p}a is quadratic residue mod p⟺∃y: y2≡a(modp)

Euler Criterion​

ap−12≡1(modp);;⇒;;a is QRa^{\frac{p-1}{2}} \equiv 1 \pmod{p} ;; \Rightarrow ;; a \text{ is QR}a2p−1​≡1(modp);;⇒;;a is QR ap−12≡−1(modp);;⇒;;a is non-QRa^{\frac{p-1}{2}} \equiv -1 \pmod{p} ;; \Rightarrow ;; a \text{ is non-QR}a2p−1​≡−1(modp);;⇒;;a is non-QR

Square Root (Special Case)​

If:

p≡3(mod4)p \equiv 3 \pmod{4}p≡3(mod4)

Then:

y≡±ap+14(modp)y \equiv \pm a^{\frac{p+1}{4}} \pmod{p}y≡±a4p+1​(modp)

Key Generation​

  • Choose primes:
p≡q≡3(mod4)p \equiv q \equiv 3 \pmod{4}p≡q≡3(mod4)
  • Compute:
n=pqn = pqn=pq
  • Public key:
nnn
  • Private key:
(p,q)(p, q)(p,q)

Plaintext Space​

x∈Zn∗x \in \mathbb{Z}_n^*x∈Zn∗​

Encryption​

y≡x2(modn)y \equiv x^2 \pmod{n}y≡x2(modn)

Decryption​

Solve:

x2≡y(modn)x^2 \equiv y \pmod{n}x2≡y(modn)

Using CRT:

x2≡y(modp)x^2 \equiv y \pmod{p}x2≡y(modp) x2≡y(modq)x^2 \equiv y \pmod{q}x2≡y(modq)

Roots​

From each:

x≡±yp+14(modp)x \equiv \pm y^{\frac{p+1}{4}} \pmod{p}x≡±y4p+1​(modp) x≡±yq+14(modq)x \equiv \pm y^{\frac{q+1}{4}} \pmod{q}x≡±y4q+1​(modq)

Total Solutions​

4 possible values of x4 \text{ possible values of } x4 possible values of x

Example​

Given:

p=7,;q=11,;n=77p = 7,; q = 11,; n = 77p=7,;q=11,;n=77

Ciphertext:

y=23y = 23y=23

Solve​

x2≡23(mod7)x^2 \equiv 23 \pmod{7}x2≡23(mod7) x2≡23(mod11)x^2 \equiv 23 \pmod{11}x2≡23(mod11)

Final Solutions​

x∈10,32,45,67x \in {10, 32, 45, 67}x∈10,32,45,67

Key Points​

  • Not injective → 1 ciphertext → 4 plaintexts
  • Requires guessing / redundancy
  • Based on square root modulo problem

Insight​

  • Uses:

    • CRT
    • Quadratic residues
    • Modular square roots

  • Limitation:

    • Ambiguity (4 solutions) → not widely used

Euler Pseudoprime​

  • (n) behaves like prime for base (a)
  • Called Euler pseudoprime

Solovay–Strassen Test​

Input:​

  • Odd integer ( n > 2 )

Steps:​

  1. Choose random:
a∈[1,n−1]a \in [1, n-1]a∈[1,n−1]
  1. Compute:
x=(an)x = \left(\frac{a}{n}\right)x=(na​)
  1. Compute:
y≡an−12mod  ny \equiv a^{\frac{n-1}{2}} \mod ny≡a2n−1​modn

Decision:​

  • If:
x=0x = 0x=0

→ (n) is composite

  • If:
y≢xmod  ny \not\equiv x \mod ny≡xmodn

→ (n) is composite

  • Else:

→ (n) is probably prime

Property​

  • If algorithm says composite → always correct
  • If prime → probabilistic

Goldwasser–Micali Cryptosystem​

Key Generation​

  • Choose primes:
p,qp, qp,q
  • Compute:
n=pqn = pqn=pq
  • Choose:
m such that (mp)=−1,;(mq)=−1m \text{ such that } \left(\frac{m}{p}\right) = -1,; \left(\frac{m}{q}\right) = -1m such that (pm​)=−1,;(qm​)=−1

Then:

(mn)=1\left(\frac{m}{n}\right) = 1(nm​)=1

✔ (m) = pseudo-quadratic residue

Public Key​

(n,m)(n, m)(n,m)

Private Key​

(p,q)(p, q)(p,q)

Encryption​

Plaintext:

x∈0,1x \in {0,1}x∈0,1

Choose random:

r∈Zn∗r \in \mathbb{Z}_n^*r∈Zn∗​

Ciphertext:

y≡r2⋅mxmod  ny \equiv r^2 \cdot m^x \mod ny≡r2⋅mxmodn

Decryption​

Check QR:

  • If:
y is QR mod p and qy \text{ is QR mod } p \text{ and } qy is QR mod p and q

→

x=0x = 0x=0
  • Else:
x=1x = 1x=1

Key Insight​

  • Security based on:

    • Difficulty of distinguishing QR vs non-QR
    • Without knowing (p, q)

Important Note​

  • Jacobi = 1 does NOT guarantee QR
  • Used to hide bit information securely

Use​

  • Probabilistic primality testing
  • Bit encryption scheme (GM cryptosystem)
